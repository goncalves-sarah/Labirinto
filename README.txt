Labirinto

Projeto da Disciplina de Programação Com Estrutura de Dados Avançados

Projeto capaz de encontrar uma saída em um labirinto, desde que o mesmo siga certas regras.

O labirinto, no formato txt, deve seguir:

O caractere de espaço indica um caminho: ' '

O caractere de asterisco indica uma parede '#'

O caractere de 'E' indica a entrada

A saída será indicada pelo caracter 'S'

Ele pode ter qualquer número de linhas ou colunas, porém, ele sempre terá forma retangular.

O labirinto deve ser inteiramente fechado com #.
